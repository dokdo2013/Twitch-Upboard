# Upboard
트위치 룰렛/슬롯머신/포인트보상 으로 나온 결과를 저장해주는 간단한 웹앱

`현재 개발 중단 상태입니다`

## 개발 동기
트윕이나 투네이션 슬롯머신/룰렛 등으로 인해 발생하는 보상을 효율적으로 관리하는 시스템이 필요하다고 판단하였음. 이에 많은 스트리머들이 사용하는 Discord에 봇을 이용해 관리하려 하였으나, 실제 제작해서 사용해본 결과 사용자 경험이 크게 부족하여 웹으로 개발하기로 결정.

## 기술 스택
- PHP CodeIgniter 3 (PHP 7.4)
- MariaDB (Amazon RDS)
- AWS Elastic Beanstalk
- Bootstrap + jQuery
